hexo的使用

2019-04-04 hexo


## hexo的安装

1. 安装node.js和git
2. 以管理员身份进入cmd.
3. 输入: npm install -g cnpm –registry=https://registry.npm.taobao.org <使用cnpm替代node.js自带的npm, 加快下载速度.
4. 输入: cnpm install -g hexo-cli

5. 下载hexo. 完后可以用hexo -v查看是否安装成功
6. 在电脑的磁盘上创建一个”Blog”文件来存放你的博客. 可以直接在我的电脑里选择一个盘,进入后在其根目录下新建一个文件夹或者用命令行实现.

7. 进入刚刚创建的文件夹.

8. 输入: hexo init ( 耐心等待一段时间, 当出现” INFO Start blogging with Hexo! “的时候就表示初始化成功了)

9. 输入: hexo s 启动博客.

10. 将上面的网址复制到浏览器或者浏览器输入localhost:4000, 就可以看到hexo的original modeul了.


## hexo部署到github

1. 在Blog文件夹目录下输入: cnpm install –save hexo-deployer-git 用来安装git部署的插件
2. 到GitHub上去建一个repository. 注意名字只能是 github账户名.github.io
3. 设置_config.yml. 在Blog目录下找到_config.yml使用vscode或者sublime等编辑器打开.按照如下配置Deployment:

4. 回到命令行,输入: git config –global user.email “你的邮箱地址”. 接着输入 git config –global user.name “你的github用户名” 来关联到账户
5. 输入: hexo d 来将hexo部署到github. 中途会让你输入你的github用户名和密码. 输入就行.

6. 成功后就可以在浏览器输入github用户名.github.io来进入自己的博客了.
7. 大功告成!
