安装hexo

  1. 安装node.js和git

  2. 以管理员身份进入cmd.

  3. 输入: npm install -g cnpm –registry=https://registry.npm.taobao.org 使用cnpm替代node.js自带的npm, 加快下载速度.

  4. 输入: cnpm install -g hexo-cli 下载hexo. 完后可以用hexo -v查看是否安装成功

  5. 在电脑的磁盘上创建一个”Blog”文件来存放你的博客. 可以直接在我的电脑里选择一个盘,进入后在其根目录下新建一个文件夹或者用命令行实现.
